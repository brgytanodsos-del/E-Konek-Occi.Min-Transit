# Security Spec

## Data Invariants
- Only Super Admins can create/update Ships, Trips, and Announcements.
- Passengers can create Ferry/Van bookings, but only if they are associated with their own account.
- Staff Admins (Port/Terminal) can update bookings and update Ship/Trip status.
- AdminAccounts can only be created/modified by Super Admin.
- Transactions are created automatically by system but managed by Admin.
